Stream Overlays

Twtich Stream overlays for charitable events and other personal streams.

For the past few years, I have been involved with a Charity Livestream on Twitchcalled “Dawn of a New Day” that raised over $8,000 for various charities, including Save the Children and the Stonewall Community Foundation. For each of the streams, I program an overlay UI that plays custom animations and holds all sorts of data from the stream. These usually interface with Stream Labels and Google App Sheets API to access the most up-to-date data. The apps have been made in Godot and Unity.

In addition to my work for this charity, I have also created an overlay for Pokemon draft streams that displays the chosen Pokemon and their stats. This tool was developed in Unity with C#. Much of the UI for these projects was generated using each tools respective markdown-like interface.
